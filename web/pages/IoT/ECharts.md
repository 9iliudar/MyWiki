# ECharts /ˈiː tʃɑːrts/

## 定义

ECharts 是前端图表库，用于构建可配置、可交互的数据可视化界面。

## 核心能力

- 多类型图表渲染
- 交互事件与联动机制
- 业务页面内嵌式可视化

## 在可视化分层中的角色

- 承担业务视角图表表达
- 强调前端定制与交互一致性
- 与 [[Grafana]] 形成“业务图表 + 监控看板”分工

## 常见误解

- 误解：ECharts 等价于完整监控平台
- 修正：监控告警与运维大盘通常由专门平台承担

## 一句话

`ECharts 是业务前端图表表达层。`
