# WebUI Style Guide

这份文档提炼当前 MyWiki WebUI 的视觉语言，便于后续复用到其他网页。

## 整体气质

这套风格的关键词是：

- 低调
- 内敛
- 克制
- 清冷
- 工程感

它不是高饱和品牌风，也不是强烈拟物风。重点在于：

- 让内容成为主角
- 让界面退到后面
- 用轻微的紫色强调建立识别，而不是主导视觉

## 配色结构

### 1. 背景层

- 主背景：深灰黑，接近 `#111111` 到 `#18181b`
- 次级背景：略亮一级的面板底，如 `var(--vp-c-bg-soft)`
- 卡片内底：比主背景稍亮，但不过分抬高对比

目标：

- 页面整体沉稳
- 层次通过明度微差体现，而不是靠大面积色块切割

### 2. 文字层

- 主文字：接近暖白的浅灰，使用 `var(--vp-c-text-1)`
- 次文字：中灰，使用 `var(--vp-c-text-2)`
- 弱信息：更淡的灰，使用 `var(--vp-c-text-3)`

目标：

- 不用纯白顶满页面
- 让阅读密度高但不刺眼

### 3. 边框层

- 边框非常轻，通常是 `var(--vp-c-divider)`
- 主要作用不是“框住”，而是轻轻划出结构边界

目标：

- 看得见层次
- 但不要让线条抢戏

### 4. 强调色

当前强调色核心是柔和紫：

- 主强调：`#8b5cf6`
- 辅助透明态：
  - `rgba(139, 92, 246, 0.35)`
  - `rgba(139, 92, 246, 0.15)`
  - `rgba(139, 92, 246, 0.06)`

这个紫色的用法非常克制，只用于：

- hover
- active 状态
- 节点高亮
- 搜索焦点
- 小范围视觉识别

不要用于：

- 大面积背景
- 大块品牌底色
- 通篇按钮填充

## 视觉语言

### 1. 组件形状

- 圆角中等偏小
- 常用半径：
  - 4px
  - 8px
  - 10px
  - 12px
  - 999px（chip）

表达的是：

- 有现代感
- 但不软萌
- 更偏冷静工具界面

### 2. 阴影

- 阴影极轻
- 主要用于 hover 或 focus
- 透明度很低，范围很窄

例子：

- `0 2px 8px rgba(139, 92, 246, 0.05)`

这说明这套风格不依赖阴影塑造层次，而依赖：

- 背景微差
- 轻边框
- 字重和留白

### 3. 排版

- 字体偏系统默认
- 字号克制，不追求 oversized hero
- 标题有重量，但不过分张扬
- 元信息明显变轻，避免和正文抢层级

层级特点：

- 标题：清晰但克制
- 辅助信息：轻
- 标签：小且不抢眼
- 卡片内容：紧凑但不拥挤

## 组件模式

### 搜索框

当前搜索框的感觉是：

- 不是显眼的输入框
- 更像一个安静的入口卡片

特征：

- 柔和背景
- 细边框
- 轻圆角
- 弱提示文字
- 右侧用轻量 `kbd` 表示快捷键

### 卡片

当前卡片风格：

- 无重阴影
- 有边框
- 有轻微背景层差
- 内容结构从上到下线性展开

卡片不是“展示型卡片”，而是“知识索引卡片”。

### 标签 / Chip

标签设计很关键：

- 小
- 瘦
- 轻
- 默认不亮

只有 active 时才使用强调色填充。

这让筛选行为有状态感，但整体仍然安静。

### 图谱

知识图谱采用：

- 深背景
- 细线连接
- 柔和紫高亮
- 小节点
- 标签文字不夸张

这让图谱更像“结构关系图”，而不是炫技动画背景。

## 交互原则

- Hover 是轻提示，不是夸张位移
- Focus 有轻微紫色描边或光晕
- Active 状态明显，但范围控制很小
- 动效偏短、偏轻，不做大幅弹跳或强过渡

## 可复用 Token 建议

如果你要把这套风格迁移到别的网页，可以先用这组变量作为起点：

```css
:root {
  --app-bg: #18181b;
  --app-bg-soft: #202127;
  --app-bg-mute: #26272f;
  --app-text-1: #f3f4f6;
  --app-text-2: #c4c7cf;
  --app-text-3: #8b90a0;
  --app-divider: rgba(255, 255, 255, 0.08);
  --app-accent: #8b5cf6;
  --app-accent-strong: rgba(139, 92, 246, 0.35);
  --app-accent-soft: rgba(139, 92, 246, 0.15);
  --app-accent-glow: rgba(139, 92, 246, 0.06);
  --app-radius-sm: 4px;
  --app-radius-md: 8px;
  --app-radius-lg: 10px;
  --app-radius-xl: 12px;
}
```

## 一句话总结

这套风格成立的关键，不是“紫色好看”，而是：

`在深色、低对比、轻边框、弱阴影的基础上，用极少量柔和紫色做精准强调。`
